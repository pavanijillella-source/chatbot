# AI Placement Preparation Chatbot

## About
This is a simple AI chatbot project developed using Python.

The chatbot can:
- Respond to user messages
- Answer basic programming questions
- Search information using Wikipedia
- Provide simple placement guidance

## Technologies Used
- Python
- Wikipedia API
- Chatbot Library

## Features
- Greeting responses
- Programming-related answers
- Wikipedia search support
- Exit/bye responses

## How to Run

Open terminal and run:

python Example.py
